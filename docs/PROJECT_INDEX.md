## 🚀 快速开始

### 新学生？（从这里开始！）
1. **首先**：阅读 [HOW_TO_START.md](HOW_TO_START.md) 了解如何开始
2. **然后**：阅读 [STUDENT_GUIDE.md](STUDENT_GUIDE.md) 了解项目
3. **接着**：阅读 [IMPLEMENTATION_GUIDE.md](IMPLEMENTATION_GUIDE.md) 了解需要实现什么
4. **参考**：查看 [CODE_TEMPLATES.md](CODE_TEMPLATES.md) 了解代码结构（**仅参考，不要复制**）
5. **规划**：参考 [WEEKLY_PLAN.md](WEEKLY_PLAN.md) 制定实现计划
6. **上手**：按照 [STUDENT_QUICKSTART.md](STUDENT_QUICKSTART.md) 快速搭建环境

### 开发者？
1. 阅读 [README.md](../README.md) 了解项目
2. 查看 [SYSTEM_DESIGN.md](SYSTEM_DESIGN.md) 理解架构
3. 参考 [ARCHITECTURE_DIAGRAM.md](ARCHITECTURE_DIAGRAM.md) 查看架构图

### 需要了解数据格式？
- 查看 [CSV_ORDER_FORMAT.md](CSV_ORDER_FORMAT.md)
